Getting Started

Prerequisites

  • Node.js (v14 or later)
  • npm or yarn
  • An AWS account (for S3 configuration)

Installation

  1. Clone the repository: 
                                    
git clone https://github.com/yourusername/nextjs-starter-repo.git
cd nextjs-starter-repo
  2. Install dependencies: 
                                    npm install
    or 
                                    yarn install
  3. Set up your environment variables (create a .env.local file)
  4. Run the development server: 
                                    npm run dev
    or 
                                    yarn dev
  5. Open your browser and navigate to http://localhost:3000

Authentication

NextAuth.js Integration

This starter uses NextAuth.js for authentication. It supports multiple OAuth providers and includes a custom user registration process.

Key Files

  • src/app/api/auth/[...nextauth]/route.ts: NextAuth API route
  • src/auth.ts: NextAuth configuration
  • src/components/Auth/Login/LoginForm.tsx: Login form component
  • src/components/Auth/Login/ProviderLogIn.tsx: OAuth provider buttons
  • src/components/Auth/Register/RegisterForm.tsx: User registration form

Authentication Flow

  1. User initiates login or registration process
  2. NextAuth.js handles OAuth provider authentication or custom credentials
  3. User session is created and managed by NextAuth.js
  4. Protected routes are secured using Next.js middleware

File Upload

AWS S3 Integration

This starter includes functionality to upload profile pictures to AWS S3.

Key Files

  • src/actions/AWS/ProfilePicUpload.ts: Handles the upload process to S3
  • src/components/Auth/Register/ImageCropper.tsx: UI for cropping and uploading profile pictures

Upload Process

  1. User selects an image for their profile picture
  2. ImageCropper component allows user to crop and adjust the image
  3. ProfilePicUpload action is called to handle the S3 upload
  4. Uploaded image URL is saved to the user's profile
